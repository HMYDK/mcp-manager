## 项目概况

- **项目名称**: MCP Manager
- **项目类型**: 开源桌面应用

- **核心目标**:

  2. 让各种 Agent CLI（Claude Code、Gemini CLI、QoderCLI 等）都能复用同一套 MCP 配置
